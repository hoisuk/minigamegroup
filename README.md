# 🎮 미니 게임 모음 (minigamegroup)

브라우저에서 바로 즐기는 가벼운 클래식 게임 5종 모음. 외부 라이브러리·서버 없이 순수 HTML·CSS·JavaScript로 만들어졌으며, 파일을 더블클릭하면 오프라인에서도 실행됩니다.

## 게임 목록

| 게임 | 파일 | 설명 |
|------|------|------|
| 💣 지뢰찾기 | `minesweeper.html` | 숫자로 지뢰 위치를 추론하는 논리 퍼즐 (초·중·고급) |
| 🔢 2048 | `game-2048.html` | 같은 숫자 타일을 밀어 합치는 슬라이드 퍼즐 |
| 🐍 스네이크 | `snake.html` | 먹이를 먹으며 길어지는 아케이드 게임 |
| 🃏 메모리 짝맞추기 | `memory.html` | 뒤집힌 카드의 짝을 찾는 기억력 게임 |
| ⭕ 틱택토 | `tictactoe.html` | 무결점 AI 대전 / 2인 로컬 대전 |

## 실행 방법

`index.html`을 브라우저로 열면 게임 선택 홈이 나타납니다. 카드를 클릭해 원하는 게임을 시작하세요.

## 특징

- **홈 허브 구조** — 홈페이지에서 게임을 골라 들어가고, 각 게임에서 홈으로 복귀
- **라이트/다크 테마** — OS 설정 자동 감지 + 수동 토글(선택값 저장)
- **다양한 입력** — 키보드 · 마우스 · 모바일 터치(스와이프) 지원, 반응형 레이아웃
- **로컬 기록** — 2048·스네이크 최고점 `localStorage` 저장
- **의존성 없음** — 빌드 과정 없이 정적 파일만으로 동작

## 라이선스
